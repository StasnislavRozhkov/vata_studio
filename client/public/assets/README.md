# Сюда кладём картинки

Положи файлы с **ровно такими именами**, и сайт подхватит их автоматически.
Если файла нет — на его месте показывается плейсхолдер в цвете образца.

## Логотип
| Файл       | Что это                                 | Статус   |
| ---------- | --------------------------------------- | -------- |
| `logo.svg` | Логотип vata (можно заменить на `.png`) | заглушка |

## Фото сумок — по цветовым вариантам

Каждая коллекция = несколько цветов. Первый вариант — основной.

**Fluffy (пушистая)** — все цвета на месте ✅
| Файл                    | Цвет     |
| ----------------------- | -------- |
| `bag-fluffy.png`        | Чёрный   |
| `bag-fluffy-gray.png`   | Серый    |
| `bag-fluffy-blue.png`   | Синий    |
| `bag-fluffy-green.png`  | Зелёный  |
| `bag-fluffy-pink.png`   | Розовый  |
| `bag-fluffy-red.png`    | Красный  |

**Standard (стандартная)**
| Файл                      | Цвет        | Статус |
| ------------------------- | ----------- | ------ |
| `bag-standard.png`        | Бежевый     | нужно  |
| `bag-standard-brown.png`  | Коричневый  | нужно  |
| `bag-standard-sand.png`   | Песочный    | нужно  |

**Metallic (металик)**
| Файл                         | Цвет     | Статус |
| ---------------------------- | -------- | ------ |
| `bag-metallic.png`           | Серебро  | нужно  |
| `bag-metallic-graphite.png`  | Графит   | нужно  |
| `bag-metallic-gold.png`      | Золото   | нужно  |

Рекомендации по фото: PNG с прозрачным фоном, высота ~1400–2000px, вес до ~1.5 МБ.

## Как добавить/изменить цвет или коллекцию

Все варианты заданы в `client/src/data/collections.ts` — поле `variants` у каждой
коллекции: `{ id, name (название цвета), swatch (цвет кружка), image (путь к фото) }`.
Добавляешь объект в `variants` и кладёшь файл — появится новый кружок-образец.
