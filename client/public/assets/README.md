## Фото сумок — `bags/<коллекция>/`

Одно фото на коллекцию. Путь к фото задаётся в
`client/src/data/collections.ts` — поле `image` у коллекции.

| Коллекция | Файл                            | Цвет на фото |
| --------- | ------------------------------- | ------------ |
| Fluffy    | `bags/fluffy/black.png`         | Чёрный       |
| Metallic  | `bags/metallic/silver.png`      | Серебро      |
| Standard  | `bags/standard/pink.png`        | Розовый      |

Рекомендации по фото: PNG с прозрачным фоном, высота ~1400–2000px, вес до ~1.5 МБ.

## Как заменить фото коллекции

Положи новый PNG в папку нужной коллекции (`bags/<коллекция>/`) и пропиши его
путь в поле `image` соответствующей коллекции в
`client/src/data/collections.ts`.
